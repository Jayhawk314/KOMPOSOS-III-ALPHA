  How Apache 2.0 Handles Patents

  Per https://www.apache.org/licenses/LICENSE-2.0:

  1. Explicit Patent Grant: Contributors grant royalty-free, irrevocable patent license for their contributions
  2. Patent Retaliation Clause: If user sues anyone over the software, their license terminates
  3. Scope Limitation: Only patents covering contributed code, not all patents
